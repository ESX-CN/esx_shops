## 下载 & 安装

### 使用 Git
```
cd resources
git clone https://github.com/ESX-CN/esx_shops [esx]/esx_shops
```

### 手动
- 下载 https://github.com/ESX-CN/esx_shops/archive/master.zip
- 解压至 `[esx]` 文件夹

## 安装
- 导入 `esx_shops.sql` 文件至你的数据库
- 添加这条至你的 `server.cfg`:

```
start esx_shops
```
